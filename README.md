# ITravel_54 — Nearby Template Removed

This build removes the Nearby section from the actual store-panel template instead of only hiding it after render. It also forces the travel/weather stat cards into a readable 2x2 layout.

## Upload
Upload all files and folders from this ZIP into the root of your GitHub repo.

## GitHub Pages
Settings → Pages → Deploy from a branch → `main` → `/root`.

## Install on phone
Open the GitHub Pages URL on Android Chrome and choose **Install app** or **Add to Home screen**.
